## Installation

Use [npm](https://www.npmjs.com) to install required dependencies

```javascript
npm i
```
## Produce a build

```bash
./build.sh -b <build_name>
```

## Run the app

```bash
./start.sh -b <build_name>
```

## Run tests

```javascript
npm run tests
```

## Pre-requisites
```

- Husky

- Docker
- Bash
```
